乾元八瑞

清代黄杨木雕八仙过海套系

此套“乾元八瑞”木雕，取材自百年黄杨，质地温润如象牙。
刀法圆熟，线条流畅，将八位仙人的神态刻画得入木三分。
时光为木色镀上一层暖金包浆，抚之如玉，观之忘俗。
这不仅是工艺的巅峰，更是那个神话与信仰共存年代的缩影。

状态：由 [ 憩园 ] 永久建档 / 离店不可逆
01

铁拐李

八仙之首，借尸还魂，法相奇特而道心通明。雕工极尽细腻，刻画其背负硕大葫芦，手扶铁拐，怒目圆睁却透出济世慈悲。衣褶随风而动，葫芦口似有灵气溢出，寓意福禄双全。

高 28cm / 黄杨木 / 附原配红木底座
02

汉钟离

坦胸露乳，手摇芭蕉扇，神情悠然自得。匠人巧妙利用黄杨木天然纹理表现其飘逸长须，双目微眯，尽显大将之风与仙家逍遥。手中宝扇似能扇去凡尘烦恼，点石成金。

高 27.5cm / 黄杨木 / 扇柄处微修 / 包浆厚重
03

张果老

倒骑毛驴心不惊，渔鼓道情醒世人。此尊造像面容沧桑，长须飘逸，不仅是长寿之相，更显看透红尘的通达。手持渔鼓，神态诙谐，木质纹理与老人皱纹完美融合，巧夺天工。饰品可以拆卸。

高 28.5cm / 黄杨木 / 渔鼓细节完好
04

吕洞宾

纯阳剑仙，风度翩翩。背负七星宝剑，手持拂尘，既有书生之儒雅，又有剑侠之英气。刀法刚柔并济，刻画出斩妖除魔的威严与浩气长存的意境，是八仙中最为俊朗的一尊。

高 29cm / 黄杨木 / 剑穗雕工极细
05

何仙姑

巧笑倩兮，步步生莲。作为八仙中唯一的女性，此作在硬朗的木材上刻画出极致的柔美。手持荷花，衣带当风，神情温婉端庄。荷花寓意出淤泥而不染，整体造型清丽脱俗。

高 26.5cm / 黄杨木 / 底部有"清赏"二字印
06

蓝采和

踏歌而行，狂放不羁。手提花篮，内盛奇花异草，寓意四季平安。神态天真烂漫，似醉非醉，于市井中参透造化玄机。衣衫褴褛中透着仙风道骨，极具艺术张力。

高 27cm / 黄杨木 / 花篮透雕工艺
07

韩湘子

紫箫吹度千波静。文质彬彬，玉树临风。轻抚长笛，似有仙乐飘渺于耳际。神情专注，尽显名士风流。木质色泽如陈年古玉，与文人气质相得益彰。

高 28cm / 黄杨木 / 箫身有轻微冰裂纹
08

曹国舅

玉板声清，官运亨通。身着官服，手执阴阳板，腰束玉带。神态端庄肃穆，不同于他仙的野逸，自有一股富贵威严之气。冠帽与官服纹饰雕刻精细，一丝不苟。

高 28.5cm / 黄杨木 / 品相完美
